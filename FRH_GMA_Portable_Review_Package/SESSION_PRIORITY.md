# Session Priority

## Best Training Order

If starting fresh, use this order:

1. Session 3
   Why:
   Mr. Smoke is one of the clearest demonstrations of Sean's implication-heavy brevity and strategic menace.

2. Session 4
   Why:
   Kobayashi shows exact wording, etiquette, compressed exposition, and criminal-mastermind control.

3. Session 5
   Why:
   Doctor Hope and Doctor Mike carry long exposition blocks that still reveal Sean's practical pacing and cosmology teaching style.

4. Session 1
   Why:
   Reagan is vital for understanding how Sean handles social derailment and novice-player sidewaysness.

5. Session 2
   Why:
   Volatile party play, abduction pressure, and early Preston integration.

6. Sessions 6 and 7
   Why:
   Strong breakout-session rhythm and old-friend player chemistry.

7. Session 8
   Why:
   Low-canon but useful table-culture texture.

## If Training For NPC Style First

Start:
- Session 3
- Session 4
- Session 5

## If Training For Table Culture First

Start:
- Session 1
- Session 2
- Session 5
