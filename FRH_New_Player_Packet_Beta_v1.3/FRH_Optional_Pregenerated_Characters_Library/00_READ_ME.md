# FRH Optional Pregenerated Characters Library

Purpose:

- provide an optional expanded shelf of FRH pregenerated characters beyond the core New Player Packet
- give players more ready-to-play choices
- give players and GMs more worked examples of successful FRH character concepts

This library is intentionally optional.

The core `FRH_New_Player_Packet_Beta_v1.3` should stay lean and onboarding-friendly.
This library can be larger, denser, and more varied.

Recommended contents:

- ready-to-play pregens
- near-final examples that are useful for customization
- short character overview notes
- `CHR`, `HTM`, and optionally `TXT` exports for each included character

Recommended exclusions:

- raw builder experiments
- failed or broken intermediate files
- redundant `_GCB` version trails once the canonical packaged version is chosen
- specimen-only engineering files that are not meaningful to players

Suggested shelves:

- `01_Ready_To_Play_Pregens`
- `02_Character_Examples`
- `03_Advanced_And_Exotic_Options`
- `04_How_To_Use_These_Characters`

Build-source note:

- `Library_Feed` subfolders are the intended beta-side source shelf for any future `beta -> release` packaging script
- those folders should contain the clean canonical player-facing source files, not the full engineering history

Packaging philosophy:

- the `NPP` is the first stop
- this library is the optional second stop
- if a player wants more range, more flavor, or more examples, this is where they go
